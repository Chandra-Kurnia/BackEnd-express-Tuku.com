# Crud app use node js & Express js


### My App

1. Node Js
2. Express Js
3. MySql

### Feature
**USER & STORE**
1. Sign Up
2. Update
3. Delete
4. Show 
5. Get all

**PRODUCT**
1. Menampilkan product dengan default limit 5
2. Custom  sort (ASC, DESC) berdasarkan waktu product dibuat
3. Custom Limit
4. Pagination
5. Search
6. CRUD (Create, Read, Update, Delete)
